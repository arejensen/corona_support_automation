============
Installation
============


Stable release
--------------

To install corona-support_automation, run this command in your terminal:

.. code-block:: console

    $ pip install corona_support_automation

This is the preferred method to install corona-support_automation, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for corona-support_automation can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/arejensen/corona_support_automation

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/arejensen/corona_support_automation/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/arejensen/corona_support_automation
.. _tarball: https://github.com/arejensen/corona_support_automation/tarball/master
